# CRUD Java Swing

Projeto CRUD desenvolvido em Java Swing.

## Funcionalidades
- Cadastro
- Consulta
- Upload de arquivos via FTP

## Tecnologias
- Java Swing
- MySQL (XAMPP)
- FileZilla Server
- Apache Commons Net

## Como executar
1. Abra o projeto no NetBeans
2. Configure o banco de dados
3. Execute a aplicação
